### ❌ 问题 2：TaskBoard 与 TaskManager 功能重叠

**设计文档中**：
```java
public class TaskBoard {
    List<Task> scanUnclaimedTasks();
    ClaimResult claimTask(int taskId, String owner);
    void updateTaskStatus(int taskId, TaskStatus status);
    Task loadTask(int taskId);
    void saveTask(Task task);
}
```

**现有代码中**（`cn.edu.agent.task.TaskManager`）：
```java
public class TaskManager {
    public synchronized Task create(...);
    public synchronized Task get(int taskId);
    public synchronized Task update(int taskId, String status, ...);
    public synchronized List<Task> listAll();
    private Task load(int taskId);
    private void save(Task task);
}
```

**冲突分析**：
- ❌ `TaskBoard` 与 `TaskManager` 功能高度重叠
- ❌ 两个类都管理任务文件的读写，违反单一职责原则
- ❌ `TaskManager` 已经是 `synchronized`，再引入 `TaskBoard` 的 `ReentrantLock` 会导致双重锁

**建议**：
1. **不要创建 TaskBoard 类**
2. **直接扩展 TaskManager**，添加以下方法：
   ```java
   // 扫描未认领任务
   public synchronized List<Task> scanUnclaimedTasks();
   
   // 认领任务（原子操作）
   public synchronized ClaimResult claimTask(int taskId, String owner);
   ```
3. 保持 `synchronized` 方法级锁，无需引入 `ReentrantLock`

### ⚠️ 问题 3：并发控制策略不一致

**设计文档中**：
- 使用 `ReentrantLock` 保护认领操作
- 扫描操作无锁（脏读）

**现有代码中**：
- `TaskManager` 所有方法都是 `synchronized`（方法级锁）
- 保证强一致性，无脏读

**冲突分析**：
- 设计文档建议的"无锁扫描"与现有的"全方法加锁"策略冲突
- 如果引入 `ReentrantLock`，需要重构整个 `TaskManager`

**建议**：
1. **保持现有的 `synchronized` 策略**（简单、可靠）
2. 性能优化可以后续考虑（当前场景下不是瓶颈）
3. 如果未来需要优化，可以考虑：
   - 读写锁（`ReentrantReadWriteLock`）
   - 文件级锁（`FileLock`）

## 七、代码示例（修正后）

### TaskManager 扩展
```java
public class TaskManager {
    // 现有方法...
    
    // 新增：扫描未认领任务
    public synchronized List<Task> scanUnclaimedTasks() throws IOException {
        return loadAll().stream()
            .filter(t -> "pending".equals(t.getStatus()))
            .filter(t -> t.getOwner() == null || t.getOwner().isEmpty())
            .filter(t -> t.getBlockedBy().isEmpty())
            .sorted(Comparator.comparingInt(Task::getId))
            .collect(Collectors.toList());
    }
    
    // 新增：认领任务
    public synchronized ClaimResult claimTask(int taskId, String owner) {
        try {
            Task task = load(taskId);
            
            // 验证认领条件
            if (task.getOwner() != null && !task.getOwner().isEmpty()) {
                return new ClaimResult(false, "Task already claimed by " + task.getOwner(), null);
            }
            if (!"pending".equals(task.getStatus())) {
                return new ClaimResult(false, "Task status is " + task.getStatus(), null);
            }
            if (!task.getBlockedBy().isEmpty()) {
                return new ClaimResult(false, "Task is blocked by " + task.getBlockedBy(), null);
            }
            
            // 原子性更新
            task.setOwner(owner);
            task.setStatus("in_progress");
            save(task);
            
            return new ClaimResult(true, "Claimed task #" + taskId, task);
        } catch (IOException e) {
            return new ClaimResult(false, "Error: " + e.getMessage(), null);
        }
    }
}
```

### ClaimResult 类
```java
public class ClaimResult {
    private final boolean success;
    private final String message;
    private final Task task;
    
    public ClaimResult(boolean success, String message, Task task) {
        this.success = success;
        this.message = message;
        this.task = task;
    }
    
    public boolean isSuccess() { return success; }
    public String getMessage() { return message; }
    public Task getTask() { return task; }
}
```
